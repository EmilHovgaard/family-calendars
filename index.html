How to subscribe

  1. Google Calendar: Other calendars → From URL → paste the link above.
  2. Apple Calendar: File → New Calendar Subscription → paste the link.
  3. Outlook: Add calendar → Subscribe from web → paste the link.
  4. Your app will refresh on its own — typically once a day — so the calendar stays up to date.

Holiday dates are read from the LFPH school calendars and may need confirming against the school's official calendar. The calendar refreshes when this feed is updated.
